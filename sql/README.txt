IS442 - G1T12 - OOP Project - Setup README

Step 1. Run deploy.sql on your machine

Step 2. Go to src/main/java

Step 3. Open DataUtility.java

Step 4. Change the port number of the CONNECTION_STRING variable at line 12 to your database port (DEFAULT WINDOWS: 3306, DEFAULT MAC : 8889)

Step 5. Comment / Uncomment line 15-16 / line 19-20 depending on database username and password configuration. Change the username and password string if necessary.

Step 6. Return to root folder, run compile.bat and run.bat to start Social Magnet.
